# Glosario
{: .fs-8 }

Un glosario bilingüe de los términos clave que aparecen en este tutorial. Los caracteres chinos se conservan junto a su equivalente en español para que puedas reconocerlos en la tabla de radicales y en los ejemplos de codificación.

## Conceptos básicos

| Chino | Español | Notas |
|---|---|---|
| 风水形音输入法 | Método de entrada Fengshui forma-fonética | El nombre de este método; «Fengshui» en abreviado |
| 风水 | Feng Shui / geomancia | El concepto tradicional que inspira la disposición |
| 字根 | radical | Los componentes básicos en que se divide un carácter |
| 字根表 | tabla de radicales | La disposición de 26 teclas de radicales |
| 字根字 | carácter raíz | Un carácter que es él mismo un radical |
| 拆字 / 拆分 | descomposición | Dividir un carácter en sus radicales |
| 编码 | código | La secuencia de teclas que produce un carácter o palabra |
| 重码 | colisión de código | Cuando un mismo código corresponde a varios caracteres |
| 候选词 / 候选 | candidato (palabra) | Una opción que ofrece el método de entrada |

## Reglas de codificación

| Chino | Español | Notas |
|---|---|---|
| 一简字 | carácter de una tecla | Teclear una tecla + Espacio para confirmar |
| 二简字 | carácter de dos teclas | Dos teclas + Espacio |
| 三简字 | carácter de tres teclas | Tres teclas + Espacio |
| 音补 / 补码 | suplemento fonético | Un código fonético que sirve como tecla final identificadora |
| 空格上屏 | pulsar Espacio para confirmar | Confirmar la selección con la barra espaciadora |
| 末码 | código final | La última tecla de un código |
| 词库 | diccionario / lista de palabras | El diccionario integrado de 157 000 palabras |
