# Установка
{: .fs-8 }

## Скачать и установить Rime

1. Скачайте с [официального сайта Rime](https://rime.im/download/) свежую версию Rime (Weasel / 小狼毫) для вашей операционной системы и установите её.

## Установить метод ввода Фэншуй

1. Скопируйте пять файлов метода ввода Фэншуй в папку установки Rime (`…\Rime\weasel-0.14.3\data`), перезаписав существующие файлы.

   Папка установки по умолчанию: `C:\Program Files\Rime\weasel-0.14.3\data`

   Пять файлов:

   - `default.yaml` — параметры метода ввода по умолчанию
   - `weasel.yaml` — параметры интерфейса метода ввода
   - `geomancy.dict.yaml` — словарь метода Фэншуй (157 000 слов)
   - `geomancy.extended.dict.yaml` — пользовательский словарь
   - `geomancy.schema.yaml` — параметры работы метода Фэншуй

2. Откройте «Пуск» → «Метод ввода Weasel» (小狼毫输入法) → «Настройки метода ввода» (输入法设定).

3. Выберите «Фэншуй форма-фонетика» (风水形音) и нажмите кнопку «中» для подтверждения.

   ![Настройки метода ввода](/assets/images/install-1.png)

4. Выберите понравившийся стиль интерфейса и нажмите кнопку «中» для подтверждения.

   ![Стиль интерфейса](/assets/images/install-2.png)

5. Нажмите одновременно <kbd>Ctrl</kbd> + <kbd>`</kbd> (клавиша в левом верхнем углу) и выберите «Фэншуй форма-фонетика» (风水形音), чтобы завершить настройку метода ввода.

   ![Смена схемы](/assets/images/install-3.png)

6. Нажмите <kbd>Ctrl</kbd> + <kbd>Shift</kbd> или <kbd>Ctrl</kbd> + <kbd>Пробел</kbd>, чтобы включить или выключить метод ввода.

   Более подробные инструкции см. в справке на [официальном сайте Rime](https://rime.im/).

## В любой момент переключиться на пиньинь

Если в какой-то момент вы не помните код ввода знака или слова, можно в любой момент переключиться в режим (<kbd>`</kbd> + пиньинь).

![Переключение на пиньинь](/assets/images/install-4.png)
