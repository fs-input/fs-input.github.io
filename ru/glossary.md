# Глоссарий
{: .fs-8 }

Двуязычный глоссарий ключевых терминов, встречающихся в этом руководстве. Китайские иероглифы приведены рядом с русскими эквивалентами, чтобы вы могли узнавать их в таблице основных знаков и в примерах кодов.

## Основные понятия

| Китайский | Русский | Примечания |
|---|---|---|
| 风水形音输入法 | Метод ввода Фэншуй «форма-фонетика» | Название этого метода ввода; сокращённо «Фэншуй» |
| 风水 | фэншуй / геомантия | Традиционное понятие, лежащее в основе раскладки |
| 字根 | основной знак / корень | Базовые компоненты, на которые раскладывается иероглиф |
| 字根表 | таблица основных знаков | Раскладка основных знаков на 26 клавишах |
| 字根字 | корневой знак | Знак, который сам является основным знаком |
| 拆字 / 拆分 | разложение | Разбиение иероглифа на его основные знаки |
| 编码 | код | Последовательность клавиш, порождающая знак или слово |
| 重码 | совпадение кодов | Когда один код соответствует нескольким знакам |
| 候选词 / 候选 | кандидат (слово) | Вариант, предлагаемый методом ввода |

## Правила кодирования

| Китайский | Русский | Примечания |
|---|---|---|
| 一简字 | одноклавишный знак | Одна клавиша + Пробел для ввода |
| 二简字 | двухклавишный знак | Две клавиши + Пробел |
| 三简字 | трёхклавишный знак | Три клавиши + Пробел |
| 音补 / 补码 | фонетическое дополнение | Фонетический код, используемый как последняя опознавательная клавиша |
| 空格上屏 | нажать Пробел для ввода | Подтвердить выбор клавишей «Пробел» |
| 末码 | последний код | Последняя клавиша кода |
| 词库 | словарь / список слов | Встроенный словарь на 157 000 слов |

{: .note }
> Пользовательские символы, которые вы видите в таблице основных знаков (например, графема после «Пример основного знака» на [главной странице](/ru/)) — это символы **зоны частного использования (PUA)** из встроенного шрифта `EUDC.ttf`. Они не входят в стандартный Unicode и отображаются только при загрузке этого шрифта.
