# 用語集
{: .fs-8 }

本チュートリアルで使われる主要用語の日中対照表です。漢字表記を日本語訳と並記することで、字根表やコード例で用語を見分けやすくしています。

## 基本概念

| 漢字 | 日本語 | 解説 |
|---|---|---|
| 风水形音输入法 | 風水形音入力メソッド | 本入力メソッドの名称。略して「風水」 |
| 风水 | 風水 | 配列の根底にある伝統的概念 |
| 字根 | 字根（じこん） | 漢字を分解した基本構成要素 |
| 字根表 | 字根表 | 26 キーの字根配列 |
| 字根字 | 字根字 | それ自身が字根である漢字 |
| 拆字 / 拆分 | 漢字分解 | 漢字をその字根に分解すること |
| 编码 | コード | 漢字や単語を入力するキーの並び |
| 重码 | 同コード / コード衝突 | 一つのコードが複数の漢字に対応すること |
| 候选词 / 候选 | 候補（語） | 入力メソッドが提示する選択肢 |
