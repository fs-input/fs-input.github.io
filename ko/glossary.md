# 용어집
{: .fs-8 }

이 강의 전체에 쓰인 핵심 용어의 이중어 대조표이다. 한자를 한국어 대응어 옆에 두어, 자근표와 코드 예에서 그것들을 식별할 수 있게 한다.

## 핵심 개념

| 한자 | 한국어 | 비고 |
|---|---|---|
| 风水形音输入法 | 풍수형음 입력법 | 이 입력법의 이름; 줄여 「풍수」 |
| 风水 | 풍수 | 배치의 기반이 되는 전통 개념 |
| 字根 | 자근 | 한 글자를 분해할 때의 기본 구성 요소 |
| 字根表 | 자근표 | 26키에 대한 자근 배치 |
| 字根字 | 자근자 | 그 자체가 하나의 자근인 글자 |
| 拆字 / 拆分 | 한자 분해 | 한 글자를 자근들로 분석 |
| 编码 | 코드 | 한 글자나 단어를 만드는 키 순서 |
| 重码 | 중복 코드 | 한 코드가 여러 글자에 대응 |
| 候选词 / 候选 | (후보) 단어 | 입력기가 제안하는 선택지 |

## 부호화 규칙

| 한자 | 한국어 | 비고 |
|---|---|---|
| 一简字 | 1키 글자 | 1키 + Space로 확정 |
| 二简字 | 2키 글자 | 2키 + Space |
| 三简字 | 3키 글자 | 3키 + Space |
| 音补 / 补码 | 음보 / 보조 코드 | 마지막 인식 키로 쓰이는 음 코드 |
| 空格上屏 | Space로 확정 | Space 키로 선택을 확정 |
| 末码 | 말코드 | 한 코드의 마지막 키 |
| 词库 | 사전 | 통합 사전 157 000 단어 |

## 문화와 배치

| 한자 | 한국어 | 비고 |
|---|---|---|
| 米人字形 / 米人布局 | 미·인 자형 / 미·인 배치 | 왼손 영역 = 「米」 자형, 오른손 영역 = 「人」 자형 |
| 阴阳 | 음양 | 중국 철학의 두 대립되는 힘 |
| 五行 | 오행 | 금(金), 목(木), 수(水), 화(火), 토(土) |
| 字母象形 | 알파벳 상형 | 알파벳이 자근의 자형과 닮음 |
| 易经 / 周易 | 역경 / 주역 | |
| 通用规范汉字表 | 통용 규범 한자표 | 8 105자 표준 세트 |

{: .note }
> 자근표에서 보이는 고유 기호(예: [홈페이지](/ko/)의 「샘플 자근」 바로 뒤의 문자)는 내장된 `EUDC.ttf` 글꼴이 그려낸 **Private Use Area(PUA)** 문자이다. 이들은 표준 Unicode가 아니며, 해당 글꼴이 로드될 때만 표시된다.
