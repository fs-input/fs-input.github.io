# Glossaire
{: .fs-8 }

Un glossaire bilingue des termes clés utilisés dans ce tutoriel. Les caractères chinois sont conservés à côté de leurs équivalents français pour que vous puissiez les reconnaître dans le tableau des radicaux et dans les exemples de codage.

## Concepts de base

| Chinois | Français | Notes |
|---|---|---|
| 风水形音输入法 | Méthode d'entrée Fengshui forme-phonétique | Le nom de cette méthode ; « Fengshui » en abrégé |
| 风水 | Feng Shui / géomancie | Le concept traditionnel qui inspire la disposition |
| 字根 | radical | Les composants de base en lesquels se divise un caractère |
| 字根表 | tableau des radicaux | La disposition à 26 touches des radicaux |
| 字根字 | caractère racine | Un caractère qui est lui-même un radical |
| 拆字 / 拆分 | décomposition | Diviser un caractère en ses radicaux |
| 编码 | code | La séquence de touches qui produit un caractère ou un mot |
| 重码 | collision de code | Quand un même code correspond à plusieurs caractères |
| 候选词 / 候选 | candidat (mot) | Un choix offert par la méthode d'entrée |

## Règles de codage

| Chinois | Français | Notes |
|---|---|---|
| 一简字 | caractère à une touche | Taper une touche + Espace pour confirmer |
| 二简字 | caractère à deux touches | Deux touches + Espace |
| 三简字 | caractère à trois touches | Trois touches + Espace |
| 音补 / 补码 | supplément phonétique | Un code phonétique servant de touche finale d'identification |
| 空格上屏 | appuyer sur Espace pour confirmer | Confirmer la sélection avec la barre d'espace |
| 末码 | code final | La dernière touche d'un code |
| 词库 | dictionnaire / liste de mots | Le dictionnaire intégré de 157 000 mots |

{: .note }
> Les symboles personnalisés que vous voyez dans le tableau des radicaux (par exemple, le glyphe qui apparaît après « Radical d'exemple » sur la [page d'accueil](/fr/)) sont des caractères de la **zone à usage privé (PUA)** tirés de la police intégrée `EUDC.ttf`. Ce ne sont pas des caractères Unicode standard et ils ne s'affichent que lorsque cette police est chargée.
