# Glossar
{: .fs-8 }

Ein zweisprachiges Glossar der Schlüsselbegriffe, die in diesem Tutorial verwendet werden. Die chinesischen Schriftzeichen stehen neben ihren deutschen Entsprechungen, damit Sie sie in der Grundzeichentabelle und in den Codebeispielen wiedererkennen.

## Grundbegriffe

| Chinesisch | Deutsch | Hinweise |
|---|---|---|
| 风水形音输入法 | Fengshui Form-Phonetic-Eingabemethode | Der Name dieser Eingabemethode; kurz „Fengshui" |
| 风水 | Feng Shui / Geomantie | Das traditionelle Konzept, das der Tastaturbelegung zugrunde liegt |
| 字根 | Grundzeichen / Radikal | Die Grundkomponenten, in die ein Zeichen zerlegt wird |
| 字根表 | Grundzeichentabelle | Die 26-Tasten-Belegung der Grundzeichen |
| 字根字 | Wurzelzeichen | Ein Zeichen, das selbst ein Grundzeichen ist |
| 拆字 / 拆分 | Zerlegung | Ein Zeichen in seine Grundzeichen zerlegen |
| 编码 | Code | Die Tastenfolge, die ein Zeichen oder Wort erzeugt |
| 重码 | Code-Kollision | Wenn ein Code mehreren Zeichen zugeordnet ist |
| 候选词 / 候选 | Kandidat (Wort) | Eine von der Eingabemethode angebotene Auswahl |
