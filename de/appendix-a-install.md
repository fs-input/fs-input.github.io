# Installation
{: .fs-8 }

## Rime herunterladen und installieren

1. Laden Sie von der [offiziellen Rime-Website](https://rime.im/download/) eine aktuelle Rime-Version (Weasel / 小狼毫) für Ihr Betriebssystem herunter und installieren Sie sie.

## Die Fengshui-Eingabemethode installieren

1. Kopieren Sie die fünf Dateien der Fengshui-Eingabemethode in den Rime-Installationsordner (`…\Rime\weasel-0.14.3\data`) und überschreiben Sie die vorhandenen Dateien.

   Standard-Installationsordner: `C:\Program Files\Rime\weasel-0.14.3\data`

   Die fünf Dateien:

   - `default.yaml` — Standardeinstellungen der Eingabemethode
   - `weasel.yaml` — Schnittstelleneinstellungen der Eingabemethode
   - `geomancy.dict.yaml` — Wörterbuch der Fengshui-Methode (157 000 Wörter)
   - `geomancy.extended.dict.yaml` — Benutzerwörterbuch
   - `geomancy.schema.yaml` — Funktionseinstellungen der Fengshui-Methode

2. Öffnen Sie „Start" → „Weasel-Eingabemethode" (小狼毫输入法) → „Eingabemethode einstellen" (输入法设定).

3. Wählen Sie „Fengshui Form-Phonetic" (风水形音) und bestätigen Sie mit der Schaltfläche „中".

   ![Eingabemethode einstellen](/assets/images/install-1.png)

4. Wählen Sie einen Ihnen zusagenden Schnittstellenstil und bestätigen Sie mit der Schaltfläche „中".

   ![Schnittstellenstil](/assets/images/install-2.png)

5. Drücken Sie gleichzeitig <kbd>Ctrl</kbd> + <kbd>`</kbd> (die Taste oben links) und wählen Sie „Fengshui Form-Phonetic" (风水形音), um die Einrichtung der Eingabemethode abzuschließen.

   ![Schema wechseln](/assets/images/install-3.png)

6. Drücken Sie <kbd>Ctrl</kbd> + <kbd>Shift</kbd> oder <kbd>Ctrl</kbd> + <kbd>Leertaste</kbd>, um die Eingabemethode ein- oder auszuschalten.

   Ausführlichere Anleitungen finden Sie in der Hilfe auf der [offiziellen Rime-Website](https://rime.im/).

## Jederzeit auf Pinyin umschalten

Wenn Sie sich zu einem Zeitpunkt nicht mehr an den Eingabecode eines Zeichens oder Worts erinnern, können Sie jederzeit in den Pinyin-Modus umschalten (<kbd>`</kbd> + Pinyin).

![Auf Pinyin umschalten](/assets/images/install-4.png)
