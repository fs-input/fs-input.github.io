# مسرد المصطلحات
{: .fs-8 }

مسرد ثنائي اللغة للمصطلحات الأساسية الواردة في هذا الدليل. وقد احتُفظ بالحروف الصينية إلى جانب مقابلها العربي لتتمكّن من التعرّف عليها في جدول الجذور وفي أمثلة الترميز.

## المفاهيم الأساسية

| الصينية | العربية | ملاحظات |
|---|---|---|
| 风水形音输入法 | طريقة إدخال فنغ شوي الشكلية-الصوتية | اسم هذه الطريقة؛ وتُختصر بـ«فنغ شوي» |
| 风水 | فنغ شوي / الجِيُومانْسيا | المفهوم التقليدي الذي يُستوحى منه التخطيط |
| 字根 | جذر (radical) | المكوّنات الأساسية التي يُفكَّك الحرف إليها |
| 字根表 | جدول الجذور | تخطيط الجذور على 26 مفتاحًا |
| 字根字 | حرف جذري | حرفٌ يكون هو نفسه جذرًا |
| 拆字 / 拆分 | التفكيك | تقسيم الحرف إلى جذوره المكوّنة |
| 编码 | الرمز | تسلسل المفاتيح الذي ينتج حرفًا أو كلمة |
| 重码 | تطابق الرموز | حين يرمّز رمزٌ واحد لأكثر من حرف |
| 候选词 / 候选 | مرشَّح (كلمة) | خيارٌ تعرضه طريقة الإدخال |

## قواعد الترميز

| الصينية | العربية | ملاحظات |
|---|---|---|
| 一简字 | حرف بمفتاح واحد | اضغط مفتاحًا واحدًا + المسافة للتأكيد |
| 二简字 | حرف بمفتاحين | مفتاحان + المسافة |
| 三简字 | حرف بثلاثة مفاتيح | ثلاثة مفاتيح + المسافة |
| 音补 / 补码 | ملحق صوتي | رمزٌ صوتي يُستعمل كمفتاح أخير للتمييز |
| 空格上屏 | اضغط المسافة للتأكيد | تأكيد الاختيار بمفتاح المسافة |
| 末码 | الرمز الأخير | آخر مفتاح في الرمز |
| 词库 | القاموس / قائمة الكلمات | القاموس المدمج المكوّن من 157 000 كلمة |

## الثقافة والتخطيط

| الصينية | العربية | ملاحظات |
|---|---|---|
| 米人字形 / 米人布局 | تخطيط رسمَي 米 (أرز) / 人 (شخص) | المنطقة اليسرى = شكل 米؛ المنطقة اليمنى = شكل 人 |
| 阴阳 | ين-يانغ | القوّتان المتضادّتان في الفلسفة الصينية |
| 五行 | العناصر الخمسة | المعدن (金)، الخشب (木)، الماء (水)، النار (火)، التراب (土) |
| 字母象形 | تصوير حرفي | حروفٌ تشبه أشكال الجذور |
| 易经 / 周易 | يي جينغ / كتاب التغيّرات | |
| 通用规范汉字表 | جدول الحروف الصينية النمطية العامة | مجموعة المعايير المكوّنة من 8 105 حرفًا |

{: .note }
> الرموز المخصّصة التي تراها في جدول الجذور (مثل الرسم الذي يظهر بعد «جذرٌ مثال» في [الصفحة الرئيسية](/ar/)) هي أحرفٌ من **منطقة الاستخدام الخاص (PUA)** مأخوذة من الخط المدمج `EUDC.ttf`. وهي ليست أحرف يونيكود قياسية، ولا تُرسم إلا عند تحميل هذا الخط.
