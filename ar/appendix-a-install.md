# التثبيت
{: .fs-8 }

## تنزيل Rime وتثبيته

1. نزّل أحدث إصدار من Rime (Weasel / 小狼毫) المناسبَ لنظام تشغيلك من [الموقع الرسمي لـ Rime](https://rime.im/download/) ثم ثبّته.

## تثبيت طريقة إدخال فنغ شوي

1. انسخ الملفات الخمسة الخاصة بطريقة إدخال فنغ شوي إلى مجلّد تثبيت Rime (`…\Rime\weasel-0.14.3\data`)، مع استبدال الملفات الموجودة.

   ومجلّد التثبيت الافتراضي هو: `C:\Program Files\Rime\weasel-0.14.3\data`

   والملفات الخمسة هي:

   - `default.yaml` — الإعدادات الافتراضية لطريقة الإدخال
   - `weasel.yaml` — إعدادات واجهة طريقة الإدخال
   - `geomancy.dict.yaml` — قاموس طريقة فنغ شوي (157 000 كلمة)
   - `geomancy.extended.dict.yaml` — القاموس الشخصي للمستخدم
   - `geomancy.schema.yaml` — إعدادات عمل طريقة فنغ شوي

2. انقر «ابدأ» ← «Weasel Input Method» (小狼毫输入法) ← «Input Method Settings» (输入法设定).

3. اختر «Fengshui Form-Phonetic» (风水形音) ثم انقر زر «中» للتأكيد.

   ![إعدادات طريقة الإدخال](/assets/images/install-1.png)

4. اختر نمطًا للواجهة يعجبك ثم انقر زر «中» للتأكيد.

   ![نمط الواجهة](/assets/images/install-2.png)

5. اضغط <kbd>Ctrl</kbd> + <kbd>`</kbd> (في أعلى اليسار) في آنٍ معًا، واختر «Fengshui Form-Phonetic» (风水形音) لإنهاء إعداد طريقة الإدخال.

   ![تغيير المخطّط](/assets/images/install-3.png)

6. اضغط <kbd>Ctrl</kbd> + <kbd>Shift</kbd> أو <kbd>Ctrl</kbd> + <kbd>Space</kbd> لتفعيل طريقة الإدخال أو إيقافها.

   لمزيد من التفاصيل عن الاستعمال، راجع مساعدة [الموقع الرسمي لـ Rime](https://rime.im/).

## التبديل إلى البينين في أي وقت

إن نسيتَ في أي لحظة رمزَ إدخال حرفٍ أو كلمة، فيمكنك التبديل في أي وقت إلى الوضع (<kbd>`</kbd> + بينين).

![التبديل إلى البينين](/assets/images/install-4.png)
